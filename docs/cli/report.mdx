# report.py

`scripts/report.py` — terminal presence report. No Slack, no LLM — just numbers in your terminal.

---

## Usage

```bash
# Specific date
python3 scripts/report.py --date 2026-03-14

# Last N days trend table
python3 scripts/report.py --trend 7

# Weekly terminal summary
python3 scripts/report.py --week
```
