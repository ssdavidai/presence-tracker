# train_model.py

`scripts/train_model.py` — manually train or retrain the three ML models on all accumulated data.

Normally runs automatically on the 1st of each month via `MonthlyMLRetrainWorkflow`.

---

## Usage

```bash
# Train on all available data
python3 scripts/train_model.py

# Force training even with <60 days of data
python3 scripts/train_model.py --force

# Train a specific model only
python3 scripts/train_model.py --model anomaly
python3 scripts/train_model.py --model recovery
python3 scripts/train_model.py --model clusters
```

## Minimum Data Requirements

| Model | Minimum days | Notes |
|-------|-------------|-------|
| Anomaly detector | 60 | Below this, use `--force`; expect high false positive rate |
| Recovery predictor | 30 | Needs enough WHOOP cycles to learn patterns |
| Focus clusters | 20 | Clusters stabilise with more data |
