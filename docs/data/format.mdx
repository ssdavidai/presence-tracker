# Data Format

Each day's data is stored as a JSONL file at `data/chunks/YYYY-MM-DD.jsonl`. One JSON line per 15-minute window — 96 lines per complete day.

---

## Field Reference

### Top Level

| Field | Type | Description |
|-------|------|-------------|
| `window_id` | string | ISO datetime of window start (no timezone) |
| `date` | string | YYYY-MM-DD |
| `window_start` | string | ISO datetime with timezone |
| `window_end` | string | ISO datetime with timezone |
| `window_index` | int | 0–95 (window 0 = 00:00, window 36 = 09:00) |

### metrics

| Field | Range | Description |
|-------|-------|-------------|
| `cognitive_load_score` | 0.0–1.0 | CLS |
| `focus_depth_index` | 0.0–1.0 | FDI |
| `social_drain_index` | 0.0–1.0 | SDI |
| `context_switch_cost` | 0.0–1.0 | CSC |
| `recovery_alignment_score` | 0.0–1.0 | RAS |

### metadata

| Field | Type | Description |
|-------|------|-------------|
| `is_working_hours` | bool | Window falls within 08:00–20:00 |
| `is_active_window` | bool | At least one source shows activity |
| `sources_available` | list | Which collectors returned data for this window |

## Working with JSONL

```python
import json
from pathlib import Path

# Read a day's data
windows = []
path = Path("data/chunks/2026-03-14.jsonl")
with open(path) as f:
    for line in f:
        windows.append(json.loads(line))

# Filter to working hours only
active = [w for w in windows if w["metadata"]["is_working_hours"]]

# Average CLS across working day
avg_cls = sum(w["metrics"]["cognitive_load_score"] for w in active) / len(active)
print(f"Average CLS: {avg_cls:.2f}")
```

## Immutability

JSONL files are treated as **immutable** once written. The pipeline appends, not updates.

To modify historical data (e.g. after a formula change):

```bash
python3 scripts/recompute_metrics.py --date 2026-03-14
```

This overwrites the file with recomputed metrics while preserving raw signals.
